# TuneCairn

TuneCairn is a personal web app for organising recordings, sheet music, notes, and tune information for traditional Scottish and Irish folk music.

This project is designed to support my music practice by keeping everything related to a tune in one place. The aim of this app is to make it easier to build a repertoire, oginise my recordings, organise tunes into sets, and connect personal resources with information from The Session.

## Features

### Planned features include:

- Store tune names and alternative titles
- Record tune types
- Upload and organise tune recordings
- Store sheet music alongside recordings
- Add practice notes and other useful information
- Group tunes into sets
- Connect tune information with data from The Session API

## Why a Web App?

TuneCairn is being built as a web app so that it can be used easily on an iPhone, desktop, or other devices without needing to publish it through the App Store or pay Apple developer fees.

## Project Goals

This is a personal project im using to learn. The main goals are to:

- Improve my web development skills
- Learn how to work with an external API
- Build a useful tool for my own traditional music practice
- Create a better way to organise and grow my tune repertoire

TuneCairn is currently in the early planning and development stage.

## Possible Future Ideas
- Search and filter tunes by type, key, or set
- Track practice progress
- Create playlists for practice sessions
- Make the app installable as a Progressive Web App

## Data Source

Tune information may be retrieved from The Session API.

The Session is a community resource for traditional Irish music and is not affiliated with this project.
